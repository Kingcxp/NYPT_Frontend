## 运行项目测试服务器
首先，你需要自行安装 git 版本控制工具、 node.js 运行环境以及 npm 包管理器。

之后，使用接下来的一连串命令来安装项目依赖：
```sh
git clone https://github.com/Kingcq/NYPT_Frontend.git
cd NYPT_Frontend
npm install
npm run dev
```
此时，你可以在浏览器访问 [http://localhost:5173](http://localhost:5173) 来查看项目运行效果。
